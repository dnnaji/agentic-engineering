# Visual Style

Use this style for generated repo visuals so principle briefs feel related.

## Shared Direction

- Original bitmap infographic, not screenshots or copied source imagery.
- Light off-white background.
- Crisp editorial technical diagram style.
- Black text with restrained blue, green, and amber accents.
- Simple line icons inside light bordered nodes.
- Clear hierarchy, generous whitespace, readable labels.
- Landscape 16:9 composition.
- No logos, watermarks, tiny text, or decorative gradients.

## Prompt Template

```text
Use case: productivity-visual
Asset type: Markdown principle brief hero/diagram for a public repository
Primary request: Create an original conceptual visual for "<principle title>".
Scene/backdrop: clean editorial technical diagram on a light off-white background.
Subject: <central system and surrounding loop/nodes>.
Style/medium: polished bitmap infographic, crisp modern editorial style, subtle line icons, technical but warm.
Composition/framing: landscape 16:9, clear hierarchy, generous whitespace, readable labels.
Lighting/mood: calm, precise, engineering-focused.
Color palette: off-white background, black text, restrained blue, green, and amber accents.
Text (verbatim): "<required labels>"
Constraints: original composition; no company logos; no copyrighted screenshots; no watermark; no tiny illegible text; keep typography professional.
```
